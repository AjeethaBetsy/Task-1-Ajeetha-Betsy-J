# Project 1 - Responsive Frontend Interface

GreenBites is a responsive restaurant website developed using HTML, CSS, and JavaScript as part of the DecodeLabs Full Stack Development Internship.

## Features
- Responsive design for mobile, tablet, and desktop
- Navigation bar with hamburger menu
- Food menu section
- CSS Grid and Flexbox layout
- Interactive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
